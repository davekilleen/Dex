# Upgrade safety platform matrix

Credential scan and rotation guidance are available wherever Python can read the vault. Local
credential migration is authorized by the complete live same-directory capability probe for
that installation, including unlisted, synced, network, removable, FAT/exFAT, and unknown-class
roots when they pass. Platform class is not an allowlist.

This Linux implementation run provides unit and local filesystem evidence only. It does not
establish native macOS 14/15 APFS, active iCloud Drive APFS interference, Windows 11/NTFS, or
other broad provider/filesystem claims. A failed capability refuses migration only and leaves
read-only scan plus manual move, validation, and rewind guidance available.
