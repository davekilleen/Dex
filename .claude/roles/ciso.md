# CISO

## Pillars

1. **Security Posture** - Defense in depth, controls, protection
2. **Risk Management** - Threat assessment, risk mitigation, business alignment
3. **Compliance** - Regulatory requirements, certifications, audits
4. **Incident Response** - Detection, response, recovery, lessons learned

## Key Workflows

- **Security Assessments** - Vulnerability management, penetration testing, audits
- **Policy Development** - Security policies, standards, procedures
- **Vendor Security** - Third-party risk, security reviews, requirements
- **Security Awareness** - Training, phishing tests, culture
- **Incident Management** - Detection, response, forensics, communication
- **Compliance Management** - SOC 2, ISO, GDPR, industry requirements

## Size Variants

### 1-100 (Startup)
- Security often handled by engineering or outsourced
- Fractional CISO or security consultant
- **Folder adjustment:** Lean structure, focus on Compliance and basics
- **Key focus:** Security foundations, SOC 2 prep, basic controls

### 100-1k (Scaling)
- First dedicated security hire or CISO
- Building security program
- Compliance requirements growing
- **Folder adjustment:** Expand Compliance, add `Awareness/`
- **Key focus:** Security program, compliance, vendor security

### 1k-10k (Enterprise)
- Full CISO role
- Security organization
- Mature program
- **Folder adjustment:** Expand all sections, add `Threat_Intel/`
- **Key focus:** Enterprise security, advanced threats, governance

### 10k+ (Large Enterprise)
- Global security organization
- SOC operations
- Board-level security
- **Folder adjustment:** Add `SOC/`, `Global/`, `Board/`
- **Key focus:** Enterprise security strategy, threat landscape, regulatory
