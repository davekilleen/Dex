# Coach

## Pillars

1. **Client Development** - Growth, transformation, goal achievement
2. **Session Excellence** - Presence, techniques, powerful conversations
3. **Practice Growth** - Marketing, referrals, business development
4. **Continuous Learning** - Skills development, certifications, supervision

## Key Workflows

- **Session Prep** - Client review, intention setting, materials
- **Client Tracking** - Progress, goals, commitments, patterns
- **Progress Documentation** - Session notes, insights, development
- **Resource Curation** - Tools, exercises, readings for clients
- **Practice Marketing** - Content, networking, visibility
- **Professional Development** - Supervision, training, certification

## Templates

- `Client_Intake.md` - New client onboarding
- `Session_Prep.md` - Pre-session preparation

- `Progress_Review.md` - Periodic progress assessment
- `Goal_Setting.md` - Client goal framework

## Coaching Type Variants

### Executive Coaching
- C-suite and senior leaders
- Leadership development
- High-stakes situations
- **Folder adjustment:** Add `Leadership/`, `Assessments/`
- **Key focus:** Leadership presence, strategic thinking, organizational influence

### Career Coaching
- Transitions and growth
- Job search support
- Professional development
- **Folder adjustment:** Add `Career_Tools/`, `Job_Search/`
- **Key focus:** Career strategy, positioning, interview prep

### Performance Coaching
- Goal achievement
- Skill development
- Accountability
- **Folder adjustment:** Add `Goals/`, `Habits/`, `Metrics/`
- **Key focus:** Goal clarity, action planning, behavioral change

### Team/Group Coaching
- Team dynamics
- Group programs
- Facilitation
- **Folder adjustment:** Add `Teams/`, `Group_Programs/`
- **Key focus:** Group dynamics, facilitation skills, program design
