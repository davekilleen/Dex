# CIO

## Pillars

1. **IT Strategy** - Technology roadmap, digital enablement, business alignment
2. **Digital Transformation** - Process automation, modernization, innovation
3. **Operations** - Infrastructure, service delivery, reliability
4. **Vendor Management** - Strategic partnerships, procurement, relationships

## Key Workflows

- **Systems Roadmap** - Enterprise architecture, application portfolio
- **Budget Planning** - IT investment, cost optimization, ROI
- **Security Posture** - Collaboration with CISO, risk management
- **Infrastructure** - Data centers, cloud strategy, network
- **Vendor Strategy** - Strategic vendors, negotiations, management
- **Digital Initiatives** - Transformation projects, automation, AI/ML

## Size Variants

### 1-100 (Startup)
- IT often handled by engineering or outsourced
- CIO role typically not present
- **Recommendation:** Consider IT Support or fractional CIO
- **Key focus:** Basic infrastructure, tool selection

### 100-1k (Scaling)
- VP IT or first CIO hire
- Building IT function
- Tool standardization
- **Folder adjustment:** Lean structure, focus on Operations and Vendors
- **Key focus:** IT foundation, security basics, tool stack

### 1k-10k (Enterprise)
- Full CIO role
- IT organization
- Digital transformation
- **Folder adjustment:** Expand all sections, add `Security/`
- **Key focus:** Enterprise IT, transformation, governance

### 10k+ (Large Enterprise)
- Global IT organization
- Complex enterprise architecture
- Major transformation initiatives
- **Folder adjustment:** Add `Global/`, `M&A/`, `Innovation_Lab/`
- **Key focus:** Enterprise technology, digital strategy, global operations
