# Dave Killeen — Personal Operating System (Pendo Field CPO)

## Why This Matters

Dave built a system nearly identical to Dex — Claude Code + MCP servers + markdown vault. He's further ahead on content intelligence (120+ newsletters, 60 YouTube channels auto-clustered) and voice input. Blueprint for features we're building.

## Key Ideas to Steal

### 1. Content Intelligence System
- Downloads YouTube transcripts from 60+ channels
- Ingests 120+ newsletters
- Auto-clusters signals across sources
- Surfaces "new and novel" + contrarian insights relevant to role
- "I don't want to be listening to 100 YouTube channels. All I do is have it say what's new and novel."

### 2. Voice Input
- Uses Whisperflow / SuperWhisper for voice-to-text
- Says voice creates more natural iteration cycles than typing
- Worth exploring for daily capture and planning

### 3. Progressive Disclosure in CLAUDE.md
- CLAUDE.md acts as navigation map, not comprehensive doc
- Points AI toward specialized files for deeper context
- Optimizes token efficiency

### 4. `/prompt-improver` Skill
- Refines vague requests into precise ones
- Prevents the #1 mistake: vagueness

### 5. Career MCP Integration
- Connects annual reviews, quarterly goals, weekly planning
- Evidence auto-compounds from meeting transcripts
- Promotion readiness scoring
- Surfaces blind spots

### 6. Rapid Prototyping Workflow
1. Idea → DEX backlog scores by impact/alignment
2. PRD creation ("10x it")
3. Taste assessment (CPO evaluates)
4. Magic Patterns → UI mockups
5. Kanban board tracks parallel streams
- "The whole guts of it was built in 37 minutes"

## Key Quote

"The new bottleneck isn't building or convincing — it's exercising taste and storytelling."

## Compounding Advantage Over ChatGPT

ChatGPT conversations live in cloud isolation. This system creates living files where:
- Meeting notes append to stakeholder pages, project files, company profiles
- Each AI interaction draws fresh context from accumulated evidence
- Cross-entity relationships strengthen automatically
